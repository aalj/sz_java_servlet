3、	基本数据类型与引用类型的区别是什么？
	答:基本数据类型在使用的时候直接在栈上分配一块内存给其使用,数据的存取都是在栈上操作
		引用数据类型在使用的时候,先在栈里面创建他的引用,引用指向堆,数据的存取操作在堆上面操作
		
4、	什么是基本数据类型的自动提升及其窄化处理?
	答:自动提升表示在使用过程中,数据类型的取值范围会向取值范围大的数据类型转化
		窄化处理 在应用过程中 根据需要把类型转化为比其取值范围更小的数据类型
5、	&与&&的区别是什么，用代码来说明
	答: 1.&可以进行位运算 && 不能进行为运算
		2.&&在使用的过程中具有短路功能
		例:
			if(1>2&&2<3){//在括号里面当运算到1>2的时候得到的值为false 以后在&&之后的运算不需要在进行运算
			}
6、	前缀++与后缀++的区别是什么，用代码说明
	答:前缀++表示先运算后赋值   后缀++表示先赋值后运算 赋值以后就可以理解为可以输出了
		例: 
			int aa = 0;
			int bb = 7;
			System.out.println(++aa);
			System.out.println(bb++);
			
		结果:
			1
			7
